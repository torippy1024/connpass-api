メンターとは

\n

Mentor (メンター)とは、英語で「指導者」という意味です。ただしメンターは、Ninja に詰め込むだけの教育をする「先生」ではありません。

\n

Ninja に自発的にセッションを進めてもらえるように隣に寄り添って補佐していただきます。

\n

メンターは授業の準備はほとんど必要ありません。 Hour of Code の内容や Scratch の基本的な操作方法が分かるかたであればどなたでもなることが出来ます。

\n
